# Portfólio Profissional - Ana Karolyne
Portfólio online pessoal-Fundamentos da Programação Web, aplicando os conceitos de HTML5, CSS3 (sem frameworks) e JavaScript.
Projeto acadêmico para a UNINTER.

## Tecnologias Utilizadas
- HTML5 (Semântica)
- CSS3 (Flexbox e Responsividade)
- JavaScript (DOM, Eventos e Validação)

## Funcionalidades
- Modo Escuro (Dark Mode)
- Validação de formulário de contato
- Layout responsivo para dispositivos móveis
